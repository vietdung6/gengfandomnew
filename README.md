### Installation

1. Clone or navigate to the project:
```bash
cd gen-g-fandom
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
